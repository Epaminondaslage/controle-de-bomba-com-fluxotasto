# controle-de-bomba-com-fluxotasto
Projeto de controle de bomba com PLC LOGO! 230R, usando fluxostato para acionamento automático. Inclui debounce do sensor, run-on para manter a bomba ligada após perda de vazão e anti-ciclo (lockout) para evitar partidas frequentes. A saída aciona um contator, garantindo proteção elétrica, confiabilidade e maior vida útil do sistema.
